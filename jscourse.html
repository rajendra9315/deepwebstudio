MODULE 12: Prototypes & Inheritance (Advanced)

Every JavaScript object has an internal prototype from which it can inherit properties and methods.

INPUT

  return "Hello " + this.name;
};

OUTPUT:

Engine Thinking

  • JS looks for method on object
  • If not found â†’ looks up prototype chain

12.2 Prototype Chain

p1 â†’ Person.prototype â†’ Object.prototype â†’ null
This lookup system is called the prototype chain.

Classes are syntax sugar over prototypes.

MODULE 18: Performance, Security & Interviews

18.1 Performance Best Practices

  • Minimize DOM manipulation
  • Debounce scroll & input events
  • Avoid blocking loops

Example: Debounce

function debounce(fn, delay) {
  let timer;
  return function() {
    clearTimeout(timer);
    timer = setTimeout(fn, delay);
  };
}
OUTPUT:
Function runs only after user stops triggering events

18.2 Security Basics

Most XSS attacks come from careless DOM insertion.
